Hyde-X Tag Shortcode

Aug 9, 2016 · 1 minute read
SnippetHugoShortcode

Want to use tags (categories) in your .md files when using the Hyde-X theme like this?

Then this line is for you:

<a class="label" href="{{ "/categories/" | absURL }}{{ .Get "txt" | urlize }}">{{ .Get "txt" }}</a>

Save in <root>/static/shortcodes/tag.html.

Use it like {{< tag txt="this" >}}.
